Web Studio Live Launches Complete Web Design in a Browser

SAN DIEGO, CALIFORNIA—Web Studio Live, a non-wizard based Web 2.0 Web Application, announces complete design control to build an entire website from the convenience and portability of a browser. Built with the freedom to customize page elements, the software unites drag and drop simplicity with the comfortable feel of a desktop application.

“We are pleased to offer this software as a stand-out option for both beginning users and experienced developers,” commented Steve Cochard, CEO, Back to the Beach Software, the makers of Web Studio Live. “This meets a real need in the market for customers searching for a convenient and limitless solution to their web design needs.”

Unlike the constraining wizard-based site builders currently in use by many hosting companies, Web Studio Live grants users complete freedom over page layouts, allowing direct visual manipulation of the page’s elements. Web Studio Live is stocked with graphic content, yet also allows the user to upload their own content, including photos, buttons, flash, videos, and others. Hosting is provided automatically with a built in hosting option, or the user is given the choice to upload to a server.

Web Studio Live also streamlines communication between professional web designers and their clients, thereby revolutionizing the review and approval process that traditionally hinders progress in constructing a website.
